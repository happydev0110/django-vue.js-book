# django-vue.js-book
django rest framework + vue 的图书管理小项目，前后端分离教程


## 环境

- Python 2.7
    - Django 1.11
    - Django Rest Framework 3.8
    
- Vue.js 2.9
    - iview 2.8
    - iview-admin 1.3

## 功能

- 简单的图书数据管理，结合豆瓣API做图书信息查询
